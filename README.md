# Vite + Tailwind + TUI Editor (React)

이 프로젝트는 **Vite + React + Tailwind CSS v3 + Toast UI Editor** 조합으로 구성된 웹 에디터 예제입니다.  
빠른 개발 경험과 유연한 스타일링, 마크다운 기반 편집 환경을 제공하는 것을 목표로 합니다.

## 핵심 기능
- **마크다운 편집기**: Toast UI Editor 기반의 편집/프리뷰 제공
- **Tailwind CSS v3**: 유틸리티 퍼스트 스타일링
- **Vite**: 빠른 번들링과 핫 리로드

## 프로젝트 구조
```
src/
  App.jsx        # 에디터 UI 및 저장 이벤트 처리
  main.jsx       # 앱 진입점
  index.css      # Tailwind 전역 스타일
```

## 실행 방법
```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 로 접속하면 확인할 수 있습니다.
